# UI/UX E-commerce Shoes App

This project is a UI/UX design assignment focused on designing
an e-commerce product details page and checkout flow for shoes.

## Screens Included
- Product Details Page
- Order Summary
- Payment Page
- Order Confirmation (Success Screen)

## Tools Used
- Figma
- MacOS Screen Recording

## UX Highlights
- Clear visual hierarchy
- Size and color variants
- Conversion-focused CTAs
- Simple and intuitive checkout flow
